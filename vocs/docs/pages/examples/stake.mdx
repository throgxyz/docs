# Staking and delegation

A Stake 2.0 walkthrough: claim pending rewards, freeze TRX for energy, and
optionally delegate that energy to another account.

To run this example:

- Clone the [tronz](https://github.com/deszhou/tronz) repository.
- Run: `TRON_PRIVATE_KEY=<key> cargo run -p examples --example stake`
