# TRC20 tokens

Read a TRC20 token's metadata and balance, then (optionally, with a key)
transfer tokens.

To run this example:

- Clone the [tronz](https://github.com/deszhou/tronz) repository.
- Run: `TRON_CONTRACT=<addr> TRON_ADDRESS=<addr> cargo run -p examples --example trc20`
