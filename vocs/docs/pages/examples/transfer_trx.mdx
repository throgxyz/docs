# Transferring TRX

Build a provider with a signer, send TRX on the Nile testnet, and wait for the
receipt.

To run this example:

- Clone the [tronz](https://github.com/deszhou/tronz) repository.
- Run: `TRON_PRIVATE_KEY=<key> cargo run -p examples --example transfer_trx`
