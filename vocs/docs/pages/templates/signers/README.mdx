## Signers

Create and use signers: generate a random keypair, sign and verify a hash with
a `LocalSigner`, derive a key from a BIP-39 mnemonic, and load or save a
Web3 Secret Storage keystore.

- [Generate a random keypair](/examples/signers/signer_generate)
- [Sign and verify a hash](/examples/signers/signer_local)
- [Derive from a BIP-39 mnemonic](/examples/signers/signer_mnemonic)
- [Encrypt and decrypt a keystore](/examples/signers/signer_keystore)
- [Sign with an AWS KMS key](/examples/signers/signer_aws)
