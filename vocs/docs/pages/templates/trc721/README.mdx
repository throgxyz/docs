## TRC721 (NFT)

Interact with TRC721 non-fungible tokens through the built-in `Trc721Instance`:
read metadata and ownership, transfer a token, and approve a single-token
spender or an operator for all tokens.

- [Query NFT metadata and ownership](/examples/trc721/trc721_query)
- [Transfer an NFT](/examples/trc721/trc721_transfer)
- [Approve a spender or operator](/examples/trc721/trc721_approve)
